# 摇摆志 使用指南

> 记录每一次选择，成就更好的决策

---

## 这是什么

摇摆志是一个个人决策记录与复盘工具。它帮你把重要的选择写下来、认真分析、做出决定，然后回头看看这个决定好不好。

简单说：**记录 → 选择 → 复盘 → 成长**。

---

## 快速开始

### 注册 / 登录

打开摇摆志，输入邮箱和密码注册账号。注册后直接登录，数据会同步到云端，换设备也能用。

### 添加到手机桌面（推荐）

- **iPhone**：用 Safari 打开 → 点底部分享按钮 → "添加到主屏幕"
- **安卓**：用浏览器打开 → 点菜单 → "添加到桌面"

添加后桌面会出现摇摆志图标，打开就像一个 App。

## 核心功能

### 1. 创建决策

点击首页的「开始新的决策」按钮。

**两种类型：**

- **快速决策**：日常小事，只需要填标题、分类和选项
- **深度决策**：重要选择，每个选项可以详细分析优点、缺点、风险、最坏结果和应对方案

**填写内容：**

- 决策标题：比如"是否跳槽到新公司"
- 分类标签：选一个预设分类，或点「+ 自定义」输入自己的分类
- 描述信息：简单说明背景（选填）
- 决策选项：至少两个，可以点「添加选项」增加更多
- 纠结程度：拖动滑杆，记录你现在有多纠结（1-5）

**使用模板：** 点右上角「模板」按钮，可以快速套用跳槽、买房、选校等常见场景的分析框架。

### 2. 做出选择

进入决策详情页，点击你最终决定的选项（支持多选），然后：

- 拖动「信心值」滑杆，记录你对这个选择有多确定（1-5）
- 点击「确认选择」

决策状态会从"进行中"变为"已完成"。

**对比视图：** 深度决策可以点右上角的对比图标，切换为左右并排对比，方便比较各选项的优缺点。

### 3. 复盘总结

过一段时间后，回到已完成的决策，进行复盘：

- 选择满意度：满意 / 一般 / 后悔
- 写下经验总结：这个决定教会了你什么
- 点击「保存复盘」

复盘完成后会收到一句温暖的反馈。

**修改复盘：** 如果想法变了，点「修改」可以重新编辑满意度和总结。

**重新打开：** 如果决定反悔了，底部有「重新打开这个决策」按钮，状态会回到进行中。

### 4. 分享复盘卡片

已完成或已复盘的决策，详情页顶部会出现分享图标。点击后：

- **保存图片**：生成一张精美的复盘卡片，保存到手机相册，可以发朋友圈或小红书
- **复制文案**：一键复制格式化的文字，直接粘贴到任何社交平台

卡片包含标题、分类、满意度、纠结度、信心值和复盘结论。底部有隐私提示，请确认内容不含敏感信息再分享。

### 5. 追加备注

已完成或已复盘的决策，底部有「追加备注」按钮。用来记录后续想到的补充内容，比如"后来又发现了一个新情况……"。

### 6. 收藏复盘

特别有价值的复盘，可以点星标收藏。在复盘中心可以通过「收藏」筛选器快速找到这些经典复盘。

## 复盘中心

点击底部导航的「复盘」进入。

### 统计概览

顶部三个数字卡片：总决策数、已完成数、已复盘数。

### 满意度分析

饼图展示所有复盘的满意度分布（满意 / 一般 / 后悔的比例）。

### 决策画像

当你有 3 条以上决策时会出现，展示：

- 你最常纠结的领域 Top 3
- 后悔率
- 平均纠结度和信心值

### 高频关键词

从你的决策标题和复盘内容中提取你反复思考的话题，帮你发现自己的关注模式。

### 搜索和筛选

- 关键词搜索
- 按状态筛选：全部 / 进行中 / 已完成 / 已复盘
- 收藏筛选
- 展开「筛选」可以按分类和时间范围（近7天 / 30天 / 3个月）进一步过滤

## 首页

### 每日提问

每天会显示一句不同的思考提问，比如"如果不考虑别人的看法，你会怎么选？"，帮你安静下来思考。

### 快速统计

四个指标卡片，每个带进度条：

- 总决策：所有记录数
- 已完成：完成率进度条
- 已复盘：复盘率进度条
- 本周新增：和上周对比的趋势箭头

### 决策质量趋势

当你有 2 条以上复盘数据时，首页会出现一条折线图，展示你每月的平均满意度变化，帮你看到自己的决策质量是否在提升。

---

## 小贴士

- 不是每个决定都需要深度分析，日常小事用「快速决策」就够了
- 复盘不用急，做完决定过几天甚至几周再回来看，效果更好
- 纠结度和信心值没有对错，诚实记录就好，它们是帮你了解自己的
- 定期看看复盘中心的决策画像，你会发现一些有趣的模式

---

## 数据安全

- 数据存储在云端（Supabase），支持跨设备同步
- 每个用户只能看到自己的数据
- 断网时可以查看缓存数据，联网后自动同步
- 不收集任何敏感信息
